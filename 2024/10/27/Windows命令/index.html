Windows CMD命令

获得mac地址

getmac /v

显示或修改文件扩展名与文件类型（文件关联）之间的映射关系

assoc .mp4

assoc .mp4=VLC.vlc

磁盘体检+修理工

命令 作用
chkdsk 只读模式，看基本信息，不修复。
chkdsk /f 修复文件系统逻辑错误（需独占时提示“下次重启再跑”）
chkdsk /f /r /f 修逻辑 + /r 扫坏道并恢复可读数据（耗时最久）。

找到进程中的脚本

tasklist | findstr script

展示wlan的报告

netsh wlan show wlanrepor

显示ip地址

netsh interface ip show address | find "IP Adress"

关闭Windows防火墙

netsh advfirewall set allprofiles state off

netsh advfirewall set allprofiles state on

查看打开的接口

netstat

关闭休眠模式

powercfg -h off
btw,这锤子模式真的消耗了C盘的大量空间

打开网络适配器

ncpa.cpl

找到使用特定端口的进程

netstat -ano | findstr ":xxxx"

字母 全称 / 意义 作用
-a all 显示 所有 连接和监听端口（不加则只显示当前活跃连接）。
-n numeric 以 数字形式 显示地址和端口号，不解析成主机名/服务名，速度更快。
-o own 显示 owning process 的 PID（进程标识符），方便后续用 tasklist 或任务管理器找进程。

获取系统所有用户

net user

账户名 类型 说明
Administrator 内置管理员 真正的超级管理员，默认被禁用（你没手动启用就不能登录）。
DefaultAccount、Guest、WDAGUtilityAccount 系统内置 都被禁用且不能登录，普通用户不用管。

WDAGUtilityAccount 是“Windows Defender Application Guard” 的虚拟账户，用来跑沙盒里的 Edge，不是人用的管理员。

获取特定用户

net user xxx

修改用户密码

net user <username> <newpassword>
