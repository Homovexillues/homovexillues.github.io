  • 方法和函数的区别是什么？

    • 一般在面向对象的语言中，这个被称为方法，面向过程语言中被称为函数，没啥太大区别，都是接收输入的数据并返回数据
    • 仅有的一点区别是：方法与其被调用的对象实例相关联，方法仅限于在其所包含的类中操作数据
    • 在C++中可以创建与对象关联的函数，被称为成员函数，在其它语言中就是方法

  • C#中[STATHread]是什么意思？

    • [STAThread]是一个特性，用于指示应用程序的默认线程是单线程单元(STA),多用在程序的入口方法上
    • STA即Single Thread Apartment，是一种线程模型，在这个模型中，每个线程都有自己的单独的消息队列，用于处理Windows消息。这种模型对于需要处理用户界面的线程非常有用，因为用户界面通常需要在单个线程上进行操作以避免并发问题
    • [STATHread]特性在使用COM Interop(托管代码与非托管代码之间的协同过程)的应用程序中非常重要。如果应用程序实际调用了COM组件，则仅为COM interop初始化该线程，如果没有使用COM interop,则不初始化该线程。

  • #region…#endregion是什么意思？

    • 在C#中，#region和#endregion是预处理指令，用于在VisualStudio中标记一块可以折叠和展开的代码区域，可以在#region后添加任意描述文字，以便记住此区域的用途，不会影响代码执行，只是提升可读性

  • ///<summary>...</summary>写法是什么意思？

    • 这是一种特殊的注释，被称为XML文档注释，这种注释可以被VisualStudio的IntelliSense（智能感知）和其他工具用来生成代码的文档

  • using关键字有什么用

    • 导入命名空间，这样就可以在代码中直接使用该命名空间中的类型，如using System后不用写System.Console.WriteLine()

  • var是什么意思

    • var是一个关键字，用于声明变量，var关键字告诉编译器，希望它根据变量的初始化表达式来推断变量类型。这被称为类型推断。
    • 如var total =0;编译器会推断出total是一个int类型的变量，因为它被初始化为一个整数。
    • var声明的变量必须在声明的时候进行初始化
    • var只能用于局部变量的声明，不能在类的级别使用var来声明字段。

  • 如果在同一类中，一个方法重写了比如，ToString(),那么同一类的方法再使用ToString的时候是会使用原有方法还是重写后的方法？

    • 如果在一个类中重写了ToString()方法，那么这个类的所有实例在调用ToString()方法时，都会使用我重写版本的方法而不是原有方法。在面向对象编程中，当一个方法被重写时，子类的方法会覆盖父类的方法。

  • [JsonIgnore]是啥？

    • 在C#中，[JsonIgnore]是一个【特性】，它来自Newtonsoft.Json库。当在一个类的属性或字段前使用[JsonIgnore]特性时，这个属性或字段将不会被序列化到JSON字符串中，也不会被从JSON字符串中反序列化。

  • 为什么一个类可以作为泛型类的类型参数来创建一个新的类？

    • 性能优化：因为泛型是值类型而不是引用类型，因此它可以避免装箱和拆箱操作
    • 具体应用方面，泛型广泛应用于集合类(如List，Dictionary<TKey,TValue>)，以及任务并行库(Task)，事件处理(EventHandler)等许多场景中。

  • 为什么属性可以被委托定义？委托会对委托封装的方法产生什么影响？委托接受的两个参数和委托定义的属性有什么关系？

    • 委托可以封装一个方法的签名，可以视作对方法的引用，因此，定义一个委托类型的属性，这个属性可以存储一个方法的引用，然后可以通过这个属性来调用这个方法。
    • 委托对其封装的方法没有直接的影响。可以将方法作为参数传递给其他方法，或者将多个方法组合在一起，此外，使用位图哦可以实现事件和回调函数。
    • 委托的参数是指通过委托调用方法时传递的参数。

  • internal static Action<string, object> Write { get; set; }是什么意思

    • internal访问修饰符，只能在同一程序集中被访问
    • static修饰符，表示这个属性属于类本身，而不是类的实例。可以通过类名来访问这个属性而不需要创建类的实例。
    • Action<string ,object>这是属性的类型。Action代表无返回值的方法的委托。这个Action委托接受两个参数，一个是string类型，一个是object类型。
    • Write属性名称
    • {get;set;}属性的访问器。
    • internal static Action<string, object> Write { get; set; }定义了一个名为Write的属性，它的类型是Action<string,object>可以在同一个程序集中被访问，且属于类本身。
    • 这个Write属性可以存储一个方法的引用，这个方法必须接受一个string参数和一个object参数，并且没有返回值。可以通过Write属性来调用这个方法。

  • 什么是回写？

    • 将数据读到内存上处理，处理之后写到硬盘上，这就是回写。

  • 什么是语法糖？

    • 语法糖是一种术语，指的是设计来使编程更容易或更可读的语言特性。这些特性并没有增加语言的功能，但是可以使代码更简洁，更易于阅读和编写。
    • 如，在C#中，属性(properties)就是一种语法糖。可以直接使用obj.Property来获取或设置值，而不需要调用obj.GetProperty或obj.SetProperty(value)。这使得代码更简洁，更像自然语言。
    • 虽然语法糖可以提高代码的可读性和编写效率，但是过度的使用可能会使代码变得难以理解和维护。
