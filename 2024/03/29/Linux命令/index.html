Linux命令

cal

输出一个简单的日历

cat

显示文件内容

cd

更改目录并进入

  • cd ..
    回到上一个目录
  • cd
    回到home文件夹

chmod

clear

清理屏幕

cmp

比较两个文件的异同

cp

复制文件到新的文件夹或者将一个文件复制到另一个文件

curl

从网络下载文件

  • curl [] -> []

diff

比较两个文件的异同并告诉我们差异之处

df

内存占用情况

du

echo

输出一些内容或向文件中添加内容

find

查找文件

free

输出内存状况

grep

  • pattern 正则表达式
  • files 要查找的文件名，可以同时查找多个文件，如果省略files参数，则默认从标准输入中读取数据

常用选项包括

  • -i 忽略大小写进行匹配
  • -v 反向查找，只打印不匹配的行
  • -n 显示匹配行的行号
  • -r 递归查找子目录中的文件
  • -l 只打印匹配的文件名
  • -c 只打印匹配的行数

head

只看文档的开头

history

查看所有执行过的历史记录

htop

用一种更花哨的方式看系统性能的进程

ip

用于管理和查看网络接口，包括物理网卡和虚拟接口

kill

结束系统进程

less

只看文档的一页

ln

链接文件

ls

man

mkdir

创建新的文件夹

more

将要查看的文件内容直接输出在终端上

mv

移动文件

neofetch

显示系统字符图像和系统信息

passwd

更改密码

  • sudo passwd nick
    修改密码

pkill

不需要知道PID

ps

查看系统进程

pwd

我在哪儿，显示目前的位置

reboot

重启

rm

删除文件

shutdown

关机，默认60s

  • shutdown now
    立刻关机

sort

将文件按字母顺序对内容进行排序

ssh

sudo

以管理员权限执行命令

su

切换用户

  • su cyber_eye
  • exit
    退出用户

systemd

涉及到操控系统方方面面的命令

tail

只看文档的末尾

touch

创建一个文件

top

查看正在消耗系统性能的进程

-U 根据用户来筛选
-k 给某一个pid一个信号,可以用来杀死进程(9)
-X 5显示完整用户名

unzip

解压缩文件

useradd/adduser

添加一个新用户/添加一个用户，可以指定密码等参数

whereis

所有命令所在文件夹

which

找到命令的位置

whoami

输出我的用户名

wget

从网络下载文件

zip

创建压缩文件

  • zip [] []
