Linux查看系统信息

这个工具属于Linux Standard Base（LSB）的一部分。LSB 是一套为 Linux 系统制定的标准化规范，旨在提高不同 Linux 发行版之间的兼容性。

选项 功能
-a 或 --all 显示所有系统信息，包括发行版名称、版本号、描述和代号
-i 或 --id 显示发行版的标识符（如 Ubuntu、Debian）
-d 或 --description 显示发行版的详细描述
-r 或 --release 显示发行版的版本号
-c 或 --codename 显示发行版的代号（如 Ubuntu 的 focal 或 Debian 的 buster）
-s 或 --short 以简洁格式输出信息，不显示标签
-h 或 --help 显示帮助信息
