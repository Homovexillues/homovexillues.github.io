Linux命令

  • ssh

  • ls

    • ls -l
      查看文件列表，包括属性
    • ls -al
      查看隐藏文件

  • pwd
    我在哪儿，显示目前的位置

  • cd
    更改目录并进入

    • cd ..
      回到上一个目录
    • cd
      回到home文件夹

  • touch
    创建一个文件

  • echo
    输出一些内容或向文件中添加内容

  • cat
    显示文件内容

  • mkdir
    创建新的文件夹

  • cp
    复制文件到新的文件夹或者将一个文件复制到另一个文件

  • mv
    移动文件

  • rm
    删除文件

    • rm -r
      删除非空文件夹

  • ln
    链接文件

  • clear
    清理屏幕

  • whoami
    输出我的用户名

  • useradd/adduser
    添加一个新用户/添加一个用户，可以指定密码等参数

  • sudo
    以管理员权限执行命令

  • su
    切换用户

    • su cyber_eye
    • exit
      退出用户

  • passwd
    更改密码

    • sudo passwd nick
      修改密码

  • which
    找到命令的位置

  • whereis
    所有命令所在文件夹

  • wget
    从网络下载文件

  • curl
    从网络下载文件

    • curl [] -> []

  • zip
    创建压缩文件

    • zip [] []

  • unzip
    解压缩文件

  • less
    只看文档的一页

  • head
    只看文档的开头

  • tail
    只看文档的末尾

  • cmp
    比较两个文件的异同

  • diff
    比较两个文件的异同并告诉我们差异之处

  • sort
    将文件按字母顺序对内容进行排序

  • find
    查找文件

    • find /bin -name “hello*”
      找到bin文件夹下所有以“hello”开头的文件

  • neofetch
    显示系统字符图像和系统信息

  • cal
    输出一个简单的日历

  • free
    输出内存状况

  • df
    内存占用情况

  • ps
    查看系统进程

    • ps -aux
      查看系统所有进程

  • top
    查看正在消耗系统性能的进程

  • htop
    用一种更花哨的方式看系统性能的进程

  • kill
    结束系统进程

    • ps -aux | grep thisbash
      显示所有进程，并抓取内容中含有“thisbash”内容的进程，然后查看process ID，也就是PID
    • kill -9 6599

  • pkill
    不需要知道PID

    • pkill -f thisbash
      直接结束含有thisbash的进程

  • systemctl

  • history
    查看所有执行过的历史记录

  • reboot
    重启

  • shutdown
    关机，默认60s

    • shutdown now
      立刻关机

  • more

    将要查看的文件内容直接输出在终端上
